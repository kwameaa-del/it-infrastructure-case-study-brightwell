# IT Infrastructure Case Study – Brightwell Design Studio

## Overview
This project is an IT infrastructure case study focused on analyzing and improving a small business environment.

The goal was to identify system performance issues, network inefficiencies, and security risks, then propose practical, cost-effective solutions.

---

## Business Scenario
Brightwell Design Studio is a small creative agency with:
- 12 employees
- 20 devices (laptops and desktops)
- Heavy use of video and audio production tools

Current environment:
- Peer-to-peer network
- Single Wi-Fi router
- Cloud-based file storage
- No centralized infrastructure management

---

## Key Problems Identified
- Slow file transfers due to cloud-only workflow and limited network design
- Weak Wi-Fi coverage from reliance on a single router
- Reduced performance during rendering due to outdated hardware
- Security vulnerability caused by unsecured wireless access

---

## Proposed Improvements
- Upgrade workstations with SSDs, increased RAM, and dedicated GPUs
- Implement a hybrid storage model (local NAS + cloud backup)
- Redesign the network using business-grade routing and multiple access points
- Improve security using WPA3, network segmentation, and multi-factor authentication

---

## Key Takeaways
- IT infrastructure directly impacts productivity in content-driven environments
- Poor system design leads to predictable performance and security issues
- Structured improvements can significantly increase efficiency and reliability
